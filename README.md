## [**Python面向对象---案例分析**](./方法综合案例)
### **目标**
1. 设计一个 `Game` 类
2. 属性:
    * 定义一个**类属性** `top_score` 记录当前游戏的历史最高分
    - 定义一个**实例属性** `player_name` 记录当前游戏的玩家姓名
3. 方法
    * **静态方法** `show_help` 显示当前游戏的帮助信息
    * **类方法** `show_top_score` 显示当前历史最高分
    * **实例方法** `start_game` 开始当前玩家的游戏
4. 主程序步骤
    * 1. 查看帮助信息
    * 2. 查看历史最高分
    * 3. 创建游戏对象, 开始游戏
